# Read me
This is the replication folder for Scaling Laws Govern Regulatory Costs from Genomes to Society


"Data" folder contains data files analyzed for the scaling of regulators (except korean companies)

"Scaling of regulators.ipynb" performs the analysis to extrapolate exponents and confidence intervals from these data. 

The "Korean company analysis" folder contains the data file and analysis code for the Korean company data, which performs data binning similar to those done for Norway companies by Statistics Norway before we acquired the data. 

"Fuction diversity analysis" folder contains the data and code needed perform the analysis in Figure 4, regarding function diversty
